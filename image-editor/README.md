Image Editor & Annotator
=========================

This is a simple browser-based image editor scaffold for LinkShare.

Features included:
- Upload image and add it to canvas
- Add editable text boxes
- Add simple emoji "stickers"
- Freehand drawing mode
- Crop with resizeable rectangle
- Rotate and flip selected objects
- Export as PNG or JPG

How to use locally:
1. Open `image-editor/index.html` in a browser (or run via local server).
2. Use the toolbar to upload an image and edit.
